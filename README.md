# 足形写真エディタ

ハンドメイド作品用に、足形スタンプや足裏写真を加工するブラウザアプリです。

## 主な機能

- 背景透過
- 選択範囲の作成と反転
- 消しゴムで透明化
- 透明にした箇所の復元
- タップした箇所に近い色の範囲選択
- ピンチ操作での拡大縮小
- 指定色への変更
- かすれや紙目を残した色変更
- 白い部分の抜き調整
- 元画像と加工後の比較
- PNG保存
- iPadのホーム画面追加向けPWA設定

## 使い方

`index.html` をブラウザで開きます。

iPadで外でも使う場合は、HTTPSで公開してからSafariで開き、共有メニューから「ホーム画面に追加」します。詳しくは `IPAD_SETUP.md` を見てください。

## プライバシー

画像処理はブラウザ内で行います。写真をサーバーへアップロードする処理は入れていません。
