# iPadで外でも使う手順

このアプリはPWAとして作ってあります。iPadのホーム画面に追加すると、普通のアプリのように起動できます。

## いちばん簡単な流れ

1. このフォルダをHTTPSで公開します。
   - おすすめは GitHub Pages、Netlify、Cloudflare Pages のどれかです。
   - iPadのオフライン起動にはHTTPSが必要です。
2. iPadのSafariで公開URLを開きます。
3. 画像を一度読み込まずに、ページが表示されるまで待ちます。
4. Safariの共有ボタンから「ホーム画面に追加」を選びます。
5. ホーム画面の「足形エディタ」アイコンから起動します。
6. 一度オンラインで起動できれば、その後は外でもアプリ本体を開けます。

## GitHub Pagesで公開する場合

1. GitHubで新しいリポジトリを作ります。
2. このフォルダの中身をアップロードします。
3. GitHubの `Settings` → `Pages` を開きます。
4. `Deploy from a branch` を選びます。
5. `main` ブランチ、`/root` を選んで保存します。
6. 表示された `https://...github.io/.../` のURLをiPadで開きます。

## Netlifyで公開する場合

1. Netlifyにログインします。
2. このフォルダをドラッグ&ドロップでアップロードします。
3. 発行された `https://...netlify.app/` のURLをiPadで開きます。

## 注意点

- 写真そのものはこのアプリ内だけで処理され、サーバーへアップロードする処理は入れていません。
- 初回だけはオンラインで開いて、ホーム画面追加まで済ませてください。
- アプリを更新した場合は、iPadで一度オンライン起動すると新しい内容がキャッシュされます。
- `file://` やUSB内のHTMLを直接開く方法では、ホーム画面アプリとしてのオフライン起動が安定しません。
